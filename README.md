# challenge-amigo-secreto
Aplicación sencilla que nos permite jugar el clásico juego del amigo secreto

# Como funciona
-- Ingresa los nombres de tus amigos en el campo indicado
-- Click en adicionar para agregarlos en la lista
-- Una vez hayas añadido a tus amigos, click en sortear amigos
-- Nuestro programa seleccionará automáticamente un nombre al azar

# Tecnologías utilizadas
-- Javascript
-- HTML
-- CSS

#Este es un proyecto libre y de uso abierto
